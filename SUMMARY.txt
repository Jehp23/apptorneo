🚀 PASOS PARA DEPLOY (20 minutos)

1️⃣ CREAR BD EN NEON (5 min)
   → neon.tech
   → Crear proyecto PostgreSQL
   → Copiar DATABASE_URL

2️⃣ VALIDAR LOCALMENTE (10 min)
   $ npm install
   $ echo "DATABASE_URL=..." > .env.local
   $ npm run db:push
   $ npm run db:seed
   $ npm run dev
   ✓ Verificar en http://localhost:3000

3️⃣ SUBIR A GITHUB (3 min)
   $ git add . && git commit -m "feat: backend"
   $ git push origin main

4️⃣ DEPLOY EN VERCEL (3 min)
   → vercel.com/new
   → Importar repo GitHub
   → Añadir DATABASE_URL en env vars
   → Deploy
